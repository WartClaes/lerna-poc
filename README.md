# Lerna

## What about it?
[Lerna](https://github.com/lerna/lerna/) is a tool for managing JavaScript projects with multiple packages in one giant mono-repo.

## The setup

```
$ npm install --global lerna@^2.0.0-beta.0
$ lerna init
```
